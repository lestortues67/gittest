Ce projet présente JSON avec les requêtes HTTP GET et POST effectuées à l'aide d'AJAX.

Le serveur attend indéfiniment une requête d'un client pour y répondre.

Le client envoie une requête de type GET (sans data) ou POST (avec data) vers le serveur.

Requête Note Serveur
GET La requête ne contient PAS de data Il se base UNIQUEMENT sur l'URL pour définir quelle data est retournée au client.
POST La requête CONTIENT de la data Il peut se baser sur l'URL OU sur la data contenue dans la requête pour définir quelle data est retournée au client.
